🔑 Was ist der API Key?

Kurze Erklärung – kein Tech-Wissen nötig

💡 Wofür brauche ich das?

Der Key ist dein persönlicher Zugangscode zur Google KI (Gemini). Die App schickt damit deine Fragen direkt an Google – komplett ohne Zwischenspeicherung auf fremden Servern.

💶 Kostet das was?
✅ Kostenlos für privaten Gebrauch

Google bietet 1.500 kostenlose Anfragen pro Tag – ohne Kreditkarte, ohne Abo. Nur ein normales Google-Konto reicht.

🚀 So holst du dir deinen Key (2 Minuten)
1
Geh auf aistudio.google.com und melde dich mit deinem Google-Konto an
2
Klick links auf „Get API key" → dann „Create API key in new project"
3
Key kopieren, links ins Feld einfügen → Speichern klicken
🔒 Ist das sicher?

Ja. Der Key wird nur in deinem Browser gespeichert – nie hochgeladen. Jederzeit mit 🗑️ löschbar.
